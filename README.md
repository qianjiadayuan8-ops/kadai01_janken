（1）制作サイトのタイトル
心理戦じゃんけんアプリ

（2）制作サイトの説明（40文字程度）
CPUの宣言を信じるか見抜くかで勝負する、心理戦要素を加えたじゃんけんアプリです。

（3）工夫した点・こだわった点
前回は仕様書通りの実装だったため、今回はオリジナル要素として心理戦を取り入れました。
 CPUが出す手を事前に宣言し、50%の確率で嘘をつく仕様にすることで、駆け引きを楽しめるよう工夫しました。
 
（4）難しかった点・次回トライしたいこと（または機能）
通常のじゃんけんロジックに加えて、CPUの宣言処理や嘘をつく判定を組み込むこと、処理の流れや設計を整理することに苦労しました。実装したい機能はいくつか思いついていたものの、ロジックが複雑になり始めた段階で整理しきれなくなり、今回は断念しました。次回は、機能を追加する前に設計を明確にしながら、より発展的な機能実装にも挑戦したいです。

（5）備考（感想、シェアしたいこと等なんでも）
思いついたアイデアを実際に実装まで落とし込めるよう、もっと時間を確保したいと思いました。
